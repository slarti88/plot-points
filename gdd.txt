Build a browser based puzzle game. The specifications of the game are as follows

1. There are x number of nodes in the game in a sequence. Each node has a number property called t. 
2. There are y number of elements. Each element can be attached to a node 
3. Each node defines which elements are allowed to be attached to that node.
4. If an element is attached to a node the same element can only be re-attached to another node after t nodes in the sequence
5. The goal is to successfully have every node attached to an element to complete the level
6. The level defines how many nodes are there and their properties and sequence and how many elements are there and their ids. 
7. Support just the single level in the first iteration

Tools
1. Use Phaser.js as the game engine
2. Use json to define the level for now

Visuals
1. For now keep it minimal with the nodes shown in the center and going from bottom to top in sequence. The nodes should be scrollable if they don't fit in the screen
2. Have the elements appear on the right of the screen and should always be visible above the nodes and not be scrollable

Interaction
1. The elements should be drag and droppable on the nodes
2. If an invalid element is added to a node it should show an indicator that it is invalid 
3. A finish button should exist to complete the level. But should be disabled if due to attaching invalid nodes or not all nodes having elements the level can't be completed
